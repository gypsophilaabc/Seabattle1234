章节1 基本规则介绍
{摆放环节}
游戏开始，双方在同样大小的方块海域上横向或纵向地摆放每一条我方船只。每条船只可以相邻，但是不能有重合的方块，也不能超出海域。
摆放环节结束。双方的布局图会被保存。接着，显示屏上出现没有敌方任何信息的海域。进入战斗环节。
{战斗环节-攻击环节}
双方同时开始攻击环节。根据己方存活的船只数（每种船只的火力不一样。后面会详细介绍），进行攻击。
双方确认输入完毕后，进入结算环节。
{战斗环节-结算环节}
显示屏上会显示击中目标或是未击中。如果一条战舰的每一个方格都被击中，那么该战舰沉没。这时，屏幕上会显示该战舰的类型和摆放位置。
{战斗环节-额外回合环节}
在进阶玩法中，一些战舰存活时具有在额外回合行动的技能。额外回合本质上是一个小的攻击回合，在这个回合中，只有这种存活的战舰可以行动。
{胜利条件判定}
以上攻击和结算环节持续进行，直至有一方最后一条战舰沉没。如果双方在同一回合所有战舰沉没，则旗舰（后面会详细介绍）生存时间最长的一方获胜。如果仍然相同，双方均失败。

章节2 武器类型介绍
海战游戏中，可以使用的武器有炮弹、鱼雷、侦察、炸弹、水雷五种。
{武器执行顺序}
在攻击回合中，执行的顺序为炮弹、鱼雷、炸弹。
在额外回合中，执行的顺序是侦察、炮弹、鱼雷、炸弹。
在摆放环节中，执行的顺序为船只、水雷。
{炮弹}
炮弹是伤害范围为1*1的一次攻击。只对水上目标造成伤害。 
{鱼雷}
鱼雷是1*4的一次攻击。但是注意：鱼雷只会对最先接触到的单元格造成一次伤害。鱼雷对水上、水下目标均可以造成伤害。但是，鱼雷不会穿过之前已经造成伤害的单元格。
{炸弹}
炸弹是伤害范围为2*2的一次攻击。只对水上目标造成伤害。
{侦察}
侦察是一次2*2的攻击。不造成伤害，但是可以获得该范围内对方战舰布局。
{水雷}
双方摆放船只完毕后，可以在地方海域投放水雷。水雷是一次2*2的伤害。对水上目标，造成一单元格伤害；对水下目标，直接沉没，并暴露其布局。
{伤害计算规则}
多次攻击同一单元格不会重复造成伤害。

章节3 舰船种类介绍
（下述的“每回合“指的是每个攻击回合。没有特殊标注额外回合的，都在攻击回合执行。）
护卫舰：规格为1*2 。存活时每回合可以发射1枚炮弹。
鱼雷艇：规格为1*2 。存活时每回合可以发射1枚鱼雷。
驱逐舰：规格为1*3 。存活时每回合可以发射1枚炮弹和1枚鱼雷。
巡洋舰：规格为1*4 。存活时每回合可以发射3枚炮弹。
重巡洋舰：规格为1*5 。存活时每回合可以发射4枚炮弹。
轻巡洋舰：规格为1*4 。存活时每回合可以发射2枚炮弹和1枚鱼雷。
战列巡洋舰：规格为2*4 。存活时每回合可以发射4枚炮弹和1枚鱼雷。
战列舰：规格为2*5 。存活时每回合可以发射6枚炮弹。
航空战列舰（1）：规格为2*4 。存活时每回合可以发射3枚炮弹。在额外回合，可以进行1次侦察。
航空战列舰（2）：规格为2*4 。存活时每回合可以发射2枚炮弹。在额外回合，可以投放1枚炸弹。
装甲战列舰：规格为2*4 。存活时每回合可以发射5枚炮弹。沉没后，在额外回合可以发射3枚炮弹。
护航航母：规格为2*3 。存活时每回合可以投放1枚炸弹。在额外回合，可以进行1次侦察。
航空母舰：规格为2*5 。存活时每回合可以投放2枚炸弹和1枚鱼雷。在额外回合，可以进行1次侦察，并投放1枚炸弹。
潜艇：规格为1*3 。存活时每回合可以投放1枚鱼雷。水下目标。
扫雷艇：规格为1*3 。存活时每回合可以发射1枚炮弹。如果开局有我方战舰触碰水雷，则牺牲扫雷艇，但暴露该区域战舰布局和类型，同时暴露扫雷艇布局；如果没有我方战舰触碰水雷，则扫雷艇存活。
岸炮：不占规格，也不受到伤害。在旗舰存活期间，在额外回合可以发射对应的炮弹数量。旗舰沉没后，已侦察区域每有1名敌方目标，可以发射1枚炮弹。
旗舰：摆放环节选定。对手可以知道旗舰的种类，但不会知道旗舰的位置和布局。旗舰存活时，岸炮可以发射总计1倍于其可发射的炮弹数或者3倍于其可投放的炸弹数的炮弹。旗舰也与胜利判定有关。请保护好您的旗舰！

章节5 游戏理念介绍
本游戏是由制作人根据经典的海战游戏规则进行扩充得到的。
本游戏不会开设任何付款购买的渠道。
制作人鼓励玩家在使用本游戏产品时，运用适当的数学方法来优化游戏表现，也鼓励在不改变游戏规则、代码的情况下，使用一定的计算机技术来辅助游玩。
本游戏旨在扩展人们对二战时期残酷的海战历史的了解。在获得战斗胜利后，您可能会解锁随机的战舰皮肤。每一款战舰皮肤都改编自二战时期真实存在过的战舰，并且会有一段它的简短的历史。在对局中，您可以向您的对手分享您使用的皮肤，并且附带有它的历史背景。
希望您在游玩本游戏的时候在得到快乐的同时，可以获得数理知识和历史知识。
